# User Guide

## Getting Started

### 1. Account Setup

1. Connect your wallet (MetaMask, WalletConnect, etc.)
2. Complete KYC verification
3. Wait for compliance approval
4. Start trading!

### 2. Placing Orders

#### Limit Orders
- Set your desired price
- Specify quantity
- Order executes when price is reached

#### Market Orders
- Execute immediately at current market price
- Best for quick trades

### 3. Managing Portfolio

- View holdings in Dashboard
- Track PnL (Profit and Loss)
- Monitor performance metrics

### 4. Using AI Signals

- AI provides trading signals based on market analysis
- Review confidence levels
- Make informed decisions

## Trading Tips

1. **Start Small**: Begin with small positions
2. **Diversify**: Don't put all eggs in one basket
3. **Use Stop-Loss**: Protect against large losses
4. **Research**: Use analytics and AI signals
5. **Risk Management**: Never trade more than you can afford to lose

## FAQ

**Q: How long does KYC take?**
A: Typically 24-48 hours, but can be faster with complete documentation.

**Q: What are the fees?**
A: Trading fees are 0.1% per trade, paid by both buyer and seller.

**Q: How does settlement work?**
A: Trades settle instantly on the blockchain using atomic swaps.

**Q: Can I withdraw anytime?**
A: Yes, withdrawals are instant after trade settlement.
